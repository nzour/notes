# Tmux

#### Common
- Scroll mode ```CTRL-A + [```

#### Session
- Detach from current session ```CTRL-A + d```

#### Window
- Create window ```CTRL-A + c```
- Drop window ```CTRL-A + &```
- Next window ```CTRL-A + n```
- Previous window ```CTRL-A + p```

#### Pane
- Create pane vertical ```CTRL-A + %```
- Create pane horizontal ```CTRL-A + "```
- Drop pane ```CTRL-A + x```
- Next pane ```CTRL-A + arrow```
